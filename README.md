# Web server en nodeJS

Realizamos un servidor web donde servimos una web de tipo red social.

## Â¿Como usarlo?

1. Debes realizar el clonado git clone https://github.com/miguelriverasarmiento/node-web-server
2. Abrir en tu editor de codigo y hacer en la terminal npm install
3. Debes armar tu variable de entorno .env con la plantilla que dejamos disponible (PORT 3000 y carpeta 'PUBLIC')
4. Ya puedes ejecutar el proyecto usando npm start
